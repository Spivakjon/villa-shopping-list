- הילדים נספרים כ-**≈4 מנות מבוגר** → סה"כ **14 מנות**
- 👶 התינוק (חודשיים) לא אוכל — מחושב כ-0, לא מופיע ברשימה
- **2 לילות** (סופ"ש)
- **2 נשים** אוכלות **סלמון** במקום בשר (לאשר אם הן טבעוניות — אם כן, להחזיר תחליפים צמחיים)
- **המנגל הוא גז** (לא פחם)

## ✨ תכונות
- ✅ **סימון ✓** בהקלקה, עם מד התקדמות כללי וספירה לכל קטגוריה
- 🔥 **מתג "על האש #1"** — מוסיף/מוריד את הכמויות שלו מהחישוב בקליק (החלטה של הרגע האחרון)
- ✎ **עריכה מלאה אונליין** — שינוי כמויות ושמות, הוספה/מחיקה של פריטים וקטגוריות
- 💾 **שמירה אוטומטית** ב-`localStorage` (לכל מכשיר בנפרד)
- 🖨️ **הדפסה / PDF** ידידותי
- 🗂️ **שחזור ברירת מחדל** + איפוס סימונים

## ⚠️ מגבלה ידועה
הסימונים והעריכות נשמרים **מקומית בכל מכשיר** (`localStorage`). אם כמה אנשים פותחים את הקישור — לכל אחד עותק משלו, **אין סנכרון חי ביניהם**. לסנכרון אמיתי צריך בקאנד (Firebase/Supabase).
